# TP_HW_1406

## Как открыть отчёт в браузере (GitHub Codespaces)

1. Запустить веб-сервер: `./run.sh report_server`
2. В Codespace открыть вкладку Ports (Порты)
3. Найти порт 8080, нажать на значок глобуса (Open in Browser)
4. Добавить к URL: `/report.html`
